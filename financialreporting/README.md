# Financial Performance Dashboard – Power BI (UI/UX Focused Reporting)

This project focuses on the design and development of a modern, executive-level Power BI dashboard, with emphasis on user experience (UX), visual storytelling, and report usability.

Unlike traditional BI projects, this solution does not include data engineering or ETL processes. The primary objective is to demonstrate advanced reporting design techniques, intuitive navigation, and clean visual composition for business users.

---

## Overview

The dashboard presents a comprehensive financial and operational performance snapshot, allowing users to quickly assess key metrics such as revenue, EBITDA, cash flow, production, and safety indicators.

The report is structured to support both high-level executive consumption and detailed analytical exploration through intuitive navigation and layered visuals.

---

## Key Features

### Executive Snapshot Design
- KPI cards with variance indicators
- Clear comparison against forecast values
- Visual emphasis on performance trends (positive vs negative)

### Multi-Page Navigation
- Tab-based navigation across key business areas:
  - Snapshot
  - Revenue
  - EBITDA
  - Production
  - Expenditure
  - Customer Margin

### Interactive Filtering
- Product-level filtering
- Channel segmentation (Domestic, Export, Intercompany)
- Time-based toggles:
  - Monthly
  - Quarterly
  - YTD
  - Full Year
  - All Time

---

## UI/UX Design Principles Applied

### 1. Visual Hierarchy
- Key KPIs positioned at the top for immediate visibility
- Supporting trends placed directly beneath
- Detailed breakdowns positioned lower in the layout

### 2. Consistent Layout
- Standardized card and chart alignment across pages
- Uniform spacing and padding
- Reusable visual patterns for familiarity

### 3. Color Semantics
- Green for positive variance
- Red for negative variance
- Neutral tones for baseline metrics

### 4. Minimal Cognitive Load
- Clean background with minimal distractions
- Focused use of labels and annotations
- Avoidance of unnecessary visuals

### 5. Contextual Navigation
- Users can move seamlessly between business areas
- Each page maintains consistent interaction patterns
