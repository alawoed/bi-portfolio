# BI Portfolio

A curated collection of end-to-end Business Intelligence solutions built on Microsoft Fabric. This portfolio demonstrates how raw data is transformed into scalable, high-performance analytics products through modern data engineering and visualization practices.

## Core Capabilities Demonstrated
- API-based data ingestion using Python  
- Lakehouse data engineering in Microsoft Fabric  
- SQL-driven data transformation (views, functions, pipelines)  
- Star schema data modeling and semantic layer design  
- Power BI (PBIP) reporting and dashboard development  
- Performance optimization and scalable architecture design  
- User-centered dashboard design and data storytelling  

## Architecture Overview
Each project follows a structured, end-to-end workflow:

1. **Data Ingestion**  
   Data is fetched from external APIs using Python and landed into the Fabric Lakehouse.

2. **Data Engineering**  
   Raw data is transformed using SQL into clean, structured tables optimized for analytics.

3. **Data Modeling**  
   A star schema is designed to support efficient querying and scalable reporting.

4. **Semantic Layer & Reporting**  
   Power BI (PBIP) is used to build semantic models, measures (DAX), and interactive dashboards.
